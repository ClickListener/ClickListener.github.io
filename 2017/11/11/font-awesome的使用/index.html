    • font-awesome的使用

    1、安装

    • 中文官网——点击下载即可下载最新版本
    • Github——Font-awesome仓库

    2、使用

    1）下载完成后【font-awesome-4.7.0.zip】，解压完成后包括以下几个文件夹：
    • CSS —— 字体的引入，图标基础样式，大小等.
    • fonts —— CSS需要引用的字体文件夹.
    • less —— 各种参数自定义的less文件，用来自定义Font awesome.
    • sass —— 各种参数自定义的sass文件，用来自定义Font awesome.

    SASS和LESS都是CSS预处理器

    2）将CSS文件夹和fonts复制到你的工程目录

    两个文件夹必须在同级同级目录，如果你想两个文件不在同级目录，比如我，想假装font-awesome是mdbootstrap自带的功能，强行给加入到mdb目录下，并将两个文件放在了不同级目录，这样的话，你需要修改font-awesome.css中的字体等文件的路径，不然运行会报错

    3）别忘了引用本地awesome
